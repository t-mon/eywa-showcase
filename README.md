# Eywa showcase

## Build instructions

In order to build this project you need at least Qt version `5.9.0`.

    $ cd eywa-showcase
    $ mkdir build
    $ cd build
    $ qmake ../
    $ make -j8
    
Finally you can run the application:

    $ ./eywa-showcase

Alternatively you can open the `eywa-showcase.pro` file with the Qt Creator and press the play button.
